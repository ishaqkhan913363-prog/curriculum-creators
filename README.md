# CurriculumAI - Professional AI-Powered Curriculum Design

An AI-powered platform for educators and curriculum designers to generate comprehensive, standards-aligned curricula with institutional-level quality and professional presentation.

## Features

- **Professional Curriculum Generation**: Create strategic curriculum frameworks suitable for educational institutions and curriculum committees
- **Standards Alignment**: Support for IB, CBSE, Common Core, Cambridge, NGSS, AP, and other international standards
- **Assessment Integration**: Generate complementary assessments and rubrics
- **Curriculum Management**: Save, view, edit, and share generated curricula
- **Real-time Collaboration**: Professional presentation with markdown formatting

## Key Capabilities

### Curriculum Design
- Executive summaries with clear learning outcomes
- Standards-aligned competency frameworks
- Unit overviews with scope and sequence
- Assessment strategies and rubrics
- Differentiation and inclusion strategies
- Resource recommendations and professional development guidance

### Assessment Generation
- Multiple assessment types (quizzes, essays, projects)
- Customizable difficulty levels
- Standards-aligned evaluation criteria
- Rubric generation for projects and performances

## Project Setup

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```
